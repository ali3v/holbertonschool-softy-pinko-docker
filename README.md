# Docker & Containerization

Learning containerization with Docker. Understanding how to build, deploy, and manage containerized applications.

## Project: Softy Pinko

Practical Docker implementation with a web application deployment scenario.

## Skills

- Docker basics and container concepts
- Creating Dockerfiles
- Container orchestration
- Image management
- Deployment workflows

## Security Context

Container knowledge matters for security:
- Container escape techniques
- Docker misconfigurations
- Securing containerized applications
- Understanding modern deployment environments
- Cloud-native security

Many targets now run in containers - understanding this technology is essential for modern pentesting.

---

Docker is increasingly common in production environments. Security professionals need to understand containerization for both offensive and defensive work.
